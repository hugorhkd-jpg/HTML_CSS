Fundamentos de HTML y CSS

¿Qué es HTML?

HTML significa HyperText Markup Language y sirve para crear la estructura de las páginas web.

Etiquetas básicas

  • <h1> Títulos
  • <p> Párrafos
  • <img> Imágenes
  • <a> Enlaces

¿Qué es CSS?

CSS (Cascading Style Sheets) se utiliza para dar estilo a las páginas web.

Funciones de CSS

  • Cambiar colores
  • Cambiar tamaños
  • Diseñar el aspecto de la web
